# Frontend design

Use this when the requested deliverable is visual UI design or UI polish.

1. Start from the user's product intent and existing visual system.
2. Make layout, spacing, typography, color, and interaction states deliberate.
3. Avoid generic styling and decorative changes that do not improve clarity.
4. Verify important states and accessibility basics before claiming completion.
