# OMP marketplace plugin activation

Use this when working with an OMP marketplace plugin package or config asset bundle.

1. Inspect packaged assets before changing behavior.
2. Prefer the OMP config tools for inventory, portability checks, and apply plans.
3. Do not publish marketplace changes or apply templates without explicit user approval.
4. Do not edit user-authored skills outside the plugin package.
5. Report missing agents, skills, hooks, commands, templates, and portability risks separately.
